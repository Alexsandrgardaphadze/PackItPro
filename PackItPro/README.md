# PackItPro

PackItPro is a secure installer packager that bundles multiple installers
into a single executable with integrity verification and controlled execution.

## Features
- Single EXE packaging
- Integrity verification
- Silent installer execution
- Elevation resume support
- Winget updater support

## Usage
1. Add installers
2. Configure options
3. Build package
4. Distribute EXE

## Safety
PackItPro does NOT modify installers.
All files are executed exactly as provided.

## Build
Requires .NET 8 SDK.
